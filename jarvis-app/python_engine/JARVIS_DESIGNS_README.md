# 🎨 JARVIS — Catálogo de Diseños `design.md`

Todos los diseños disponibles para instalar en tus proyectos via `npx designdotmd`.

## 🚀 Cómo usar

```
# En modo Programador, con proyecto activo:
Jarvis, instalá el diseño terminal
Jarvis, usá el diseño cyberpunk
Jarvis, aplicá el estilo matcha

# Al crear un proyecto nuevo, se auto-detecta:
"quiero que programes una cafeteria con estilo linen"
```

El archivo `DESIGN.md` se instala en la carpeta del proyecto y **qwen2.5-coder lo lee antes de generar cualquier código**, garantizando que la tipografía, colores y tokens sean 100% consistentes.

## ⚡ Referencia rápida — Comando de instalación

```bash
# Instalar directamente via CLI (en la carpeta del proyecto):
npx designdotmd add terminal
npx designdotmd add matcha
npx designdotmd add cyberpunk-city
npx designdotmd add sports-hud

# Ver categorías disponibles:
npx designdotmd category

# Ver todos los diseños:
npx designdotmd list
```
